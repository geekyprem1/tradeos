# TradingOS

**TradingOS** is a Behavior Change System (a Trading Discipline Operating System) that acts as a proactive psychological layer between the trader's mind and the market, engineering consistent execution of a defined edge by making disciplined trading the path of least resistance.

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set up environment variables:
   Copy `.env.example` to `.env.local` and add your keys (Supabase, Resend, VAPID, etc.).

3. Run development server:
   ```bash
   npm run dev
   ```
